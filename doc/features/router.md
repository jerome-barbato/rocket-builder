# Router

###### file: js/metabolism/router.js
    
Simple hash router

~~~~    
<section>
  <article page="home" default>
        Home
    </article>
    <article page="page-1">
        Page-1
    </article>
    <article page="page-2">
        Page-2
    </article>
</section>

<nav>
   <a href="#/home">Home</a>
   <a href="#/page-1">Page 1</a>
   <a href="#/page-2">Page 2</a>
</nav>
~~~~

Define animation using meta
~~~~
<meta name="animation-router" content="slide/fade">
~~~~
