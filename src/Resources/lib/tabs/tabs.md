# Tabs
   
######file: js/metabolism/toggle.js

Simple tab management system

~~~~
<div tabs>
    <nav>
        <a href="#1">Item 1</a>
        <a href="#2">Item 2</a>
    </nav>
    <div id="1">
        Tab Item 1
    </div>
    <div id="2">
        Tab Item 2
    </div>
</div>
~~~~

### Parameters

You can transform tab to inline insert

~~~~
<div tabs data-inline="mobile">
    <nav>
        <a href="#1">Item 1</a>
        <a href="#2">Item 2</a>
    </nav>
    <div id="1">
        Tab Item 1
    </div>
    <div id="2">
        Tab Item 2
    </div>
</div>
~~~~

Available parameters are mobile, tablet, phone
